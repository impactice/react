## AI와 효과적으로 대화하기 
**좋은 답변을 위한 다섯 가지 원칙** 
- 구체적 지시: AI는 사용자의 마음 속을 들여다볼 수 없다 
- 명확한 단어: 인공지능의 학습 과정을 고려해서, 인공지능이 이해할 수 있는 명확한 단어 사용 
- 맥락 제공: 설명하기 복잡한 경우 예시나 맥락을 제공하면 좋다
- 단계별 접근: 인공지능은 '생각'을 하는 것이 아니므로 적절한 추론 형식과 도출 과정 제시 
- 일관성 유지: 마음에 드는 답변을 받았다면, 프롬프트를 '템플릿'처럼 사용하자

# 챗봇 AI별 특징 알아보기

챗봇 AI는 각기 다른 기능과 특징을 지니고 있으며, 사용 목적과 환경에 따라 적절한 모델을 선택하는 것이 중요합니다. 아래는 주요 챗봇 AI의 기능 및 특징 요약입니다.

---

## 📌 공통 기능 요약

- **파일 업로드 및 질문**: 문서를 기반으로 질의응답 가능
- **음성 입력 및 출력**: 말로 입력하고 음성으로 답변 가능
- **사용자 성향 파악**: 메모리를 통해 사용자 맞춤형 응답 제공 (원할 경우 설정 가능)
- **답변의 출처 표기**: AI 검색 기능을 통해 근거 있는 답변 제공
- **멀티모달 AI 지원**: 텍스트 외 이미지, 음성 등 다양한 입력 형식 지원 (일부 AI만)

---

## 🤖 주요 AI 모델 특징

### ChatGPT (OpenAI)
- **모델 선택 가능** (유료 포함) 
- 파일 업로드해서 질문을 할 수 있음
- 음성 답변 제공
- 검색을 클릭하면 AI검색 기능을 사용하는 것임 
- 이성을 클릭하면 추론 모델, 시간이 많이 걸린다. 하지만 구체적이고 정확한 답변을 받을 수 있다 
- ChatGPT 맞춤 설정으로 들어가면 호칭, 특성을 설정할 수 있음 
- 설정에서 개인 맞춤 설정은 나에 대해 파악하는 것임(나를 파악하는 것이 싫으면 끄면 됨) 
- 나에 대해 파악하는 것은 메모리 관리하기에 있음 (원하지 않은 메모리 부분은 삭제 할 수 있다) 
- 데이터 제어에 있는 모두를 위한 모델 개선은 내가 입력한 내용을 학습 활용에 동의함을 의미하는 데 끄는 것을 **추천** 
- 필더 프로필은 개발자나 챗봇을 구축하는 사람들과 관련있다 
- 연결된 앱은 연결된 사이트에 있는 파일을 쉽게 불러와서 사용 할 수 있다
---

### Gemini (구글) 
- 모델 선택 가능 
- 파일 업로드해서 질문을 할 수 있음 
- 음성 답변 제공 
- 확장 프로그램 사용 가능 (구글의 장점을 최대한 활용 가능) 
- 메일의 내용을 요약하거나 번역, 구글 드라이브등에 있는 파일을 질문에 활용한다 
- 구글 지도, 항공편, 호텔에서 정보를 가져와서 알려준다 
- 유튜브나 음악을 가져와서 답변한다
- 구글 정보와 연동하여서 제공해준다
--- 

### Claude (클로드) 
- 한국어를 잘한다 
- 이미지 생성은 못하지만 텍스트 생성과 코딩에 최적화 되어있다 
- 설정의 프로필에 호칭을 정할 수 있다 
- 코딩을 한 것을 실행도 해 볼 수 있다
- 구체적인 한국어 텍스트를 생성할 때 많이 활용이 되고 있다 
--- 

### 클로바 X (네이버)
- 네이버 서비스와 강력한 연동 (예: 네이버 쇼핑, 지도) 
- 정보 탐색 도우미는 AI 검색과 비슷하다
- 문서 업로드 및 대화형 질문 가능 
- AI 지우개는 멀티 모달 AI
- 스킬 기능을 통해 특정 서비스와 연동된 답변 제공
---

### 에이닷

# AI와 실무 문서 작성 
## 비즈니스 이메일 예시
- 특정한 역할을 주는 페르소나
너는 비즈니스 커뮤니케니션 전문가야

- 과업 핵심적인 업무 내용
거래처에서 우리 회사에서 새로 출시한 보안 소프트웨어를 소개하는 비즈니스 이메일을 작성해줘

- 맥락 - 배경이나 목적
이 이메일은 거래처에 새로운 제품의 주요 기능과 장점을 소개하고, 관심을 유도하여 미팅이나 데모 요청으로 이어질 수 있도록 해야 해. 이메일은 상대방의 시간을 고려해서 간결하면서도 핵심 내용을 담아야 하며, 제품의 독창성과 경쟁 우위를 강조해야 해. 이메이를 끝에는 문의나 데모 요청을 위한 연락처나 링크를 포함해줘.

- 어조

---

### 💥 강력한 도전에 직면한 구글

- AI가 생성하는 저품질 데이터가 인터넷에 급증하면서 **할루시네이션**과 허위 정보 증가
- 샘 알트먼:  
  “우리는 검색을 훨씬 더 개선할 여지가 있다고 생각한다”  
  “이것이 구글 검색 방식보다 훨씬 더 좋다는 점과 제가 얼마나 빨리 적용했는지에 놀랐다”고 발언
- 서치GPT는 검색 엔진과 구글을 정면 겨냥한 제품으로 AI 검색 엔진을 지향하는 **퍼플렉시티AI**에도 영향을 끼칠 수 있음
- 서치GPT의 비즈니스 모델이 독특한 것은 아니기 때문에, **비즈니스가 검증되면 다른 모델 기업들이 따라갈 것으로 예상됨**

---

### 🚀 달아오르는 AI 검색 시장

- 2024년 4월 기준 **퍼플렉시티AI의 검색 서비스 사용자 수**는 약 **1,500만 명**,  
  월간 방문자 수는 **5,000만 명**에 달할 정도로 AI 기반 검색에 대한 시장 반응은 달아오르고 있음
- **직관적 사용자 경험(UX)** 을 제공한다면 서치GPT가 검색 시장을 크게 흔들 수 있음
- 서치GPT, 퍼플렉시티 같은 AI 애플리케이션이 마치 아마존, 애플뉴스, 마이크로소프트 스타트페이지처럼  
  **여러 콘텐츠 제공자들의 콘텐츠를 모아서 보여주는 뉴스 애그리게이터(Aggregator, 통합제공자)** 로 자리잡게 되면서  
  **공급원인 언론사, 퍼블리셔, 콘텐츠 제작자들의 영향력과 가치**가 더 커질 수 있음

- 효율적인 협업: 실시간 피드백 및 팀원과의 작업 공유 가능 
- 유연성: 다양한 템플릿과 스타일을 즉시 변경 가능

- Stop 4: 피드백 및 수정 
 - 완성된 슬라이드 검토 및 사용자 경험을 반영한 수정

# AI로 이미지 생성하기 
## 완성도를 좌우하는 핵심 요소 이해 
- 6대 핵심 요소: 주제, 맥락, 생상, 텍스처와 스타일, 배경, 반복과 수정 
- 전달력 강화: 이미지는 단순한 장식이 아니라 메시지를 전달하는 도구 
6요소를 고려하면 명확하고 일관된 메시지를 표현할 수 있음 
- 감성적 연결과 매력 증가: 잘 구성된 이미지가 보는 사람의 감정에 더 큰 영향을 미침 
색상, 텍스처, 배경 등은 사람들의 공감을 이끌어내는 데 중요한 역할을 함 
- 창의성과 효율성의 균형: 6요소를 활용하면 창의적이면서도 체계적인 결과물을 빠르게 제작 가능. AI와 결합하면 초보자도 전문가 수준의 작업 가능 

## 이미지의 핵심 메시지를 결정하는 '주제' 
- 🖼️ 왜 중요한가?
  - 이미지의 목적과 메시지를 결정하는 첫 단계
  - 구체적이고 명확한 주제가 있어야 일관성 있는 이미지 생성 가능

- 🎯 주제를 어떻게 설정할까?
  - 자신에게 물어보자:  
    _“이 이미지로 무엇을 말하고 싶은가?”_
  - 주제와 관련된 키워드를 구체적으로 도출해 보자

- 📝 주제 키워드 예시
  - 활기찬 도시의 야경  
  - 낮잠을 자는 고양이  
  - 모던한 인테리어의 사무실 등

## 이미지의 메시지와 흐름을 결정하는 '맥락'
- 맥락이란 무엇인가?
  - 맥락은 이미지의 요소(구성)와 배경 이야기(상황)를 포함한 큰 그림
  - 시청자가 이미지의 의도를 더 쉽게 이해하고 공감할 수 있도록 도움
  - 맥락은 '구성'과 '상황'을 모두 고려해 제시해야 함

- 구성
  - 맥락 안에서 요소들이 어떻게 배치되는지 결정
  - 예: "산 위에 큰 성이 있고, 그 앞에 나무들이 둘러싸여 있다."

- 상황
  - 맥락 안에서 이미지가 '어떤 장면'인지 구체화
  - 시간, 장소, 분위기, 감정 등의 디테일을 설정
  - 예: "어두운 밤, 캠프파이어 옆에서 이야기를 나누는 사람들"

## 이미지의 분위기를 결정하는 '색상' 
- 감정과 분위기 형성
  - 색상은 이미지가 전달하는 감정을 좌우함
  - 예: 빨강은 열정과 에너지, 파랑은 안정감과 신뢰감을 상징

- 시각적 집중 유도
  - 특정 색상이 보는 사람의 시선을 끌고, 주제 강조에 도움
  - 예: 단색 배경에서 눈에 띄는 밝은 색의 오브젝트

- 브랜드 아이덴티티 표현
  - 일관된 색상 팔레트는 브랜드의 이미지를 강화
  - 예: 코카콜라의 빨강, 스타벅스의 녹색

## 이미지에 깊이를 더하는 '텍스처'와 '스타일' 
- 텍스처의 역할
  - 평면적인 이미지를 생동감 있게 만듦
  - 현실감과 디테일을 강화

- 텍스처를 효과적으로 사용하는 방법
  - 강조할 때: 물리적인 소재(예: 천, 나무)를 묘사하거나, 초현실적인 분위기를 만들 때 사용
  - 단순화할 때: 주제 강조를 위해 불필요한 텍스처를 제거
  - AI 입력 예시: "부드러운 직물 텍스처의 소파"

- 스타일이 중요한 이유
  - 이미지의 전체 분위기와 메시지를 좌우함
  - 특정 스타일은 보는 사람에게 더 강렬한 인상을 줄 수 있음
  - 예: 수채화는 부드럽고 감성적인 분위기, 3D 렌더링은 미래적이고 사실적인 느낌

## 주제를 돋보이게 하는 '배경' 
- 배경의 역할
  - 주제를 보완하거나 강조
  - 산만하지 않으면서도 시선을 끌 수 있어야 함

- 배경 선택 시 고려 사항
  - 단순 배경 vs 복잡 배경의 장단점
  - 색상과 조화로운 배경 설정 팁

## 완벽한 이미지를 위한 '반복과 수정' 
- 반복 작업의 중요성
  - 첫 시도에서 완벽한 결과물이 나오는 경우는 드묾
  - 반복적으로 수정하며 최적화 필요

- AI로 반복 작업 효율화
  - AI는 사용자의 피드백을 즉각 반영해 수정 가능
  - 여러 버전을 생성해 비교 선택 가능
  - 일부 AI 서비스에서는 특정 부분만 선택해서 수정할 수 있는 기능 제공

## 6요소를 잘 활용하려면  
<프롬프트 작성의 기본 규칙 만들기>
 - 6가지 요소(주제, 맥락, 색상, 텍스처와 스타일, 배경, 반복과 수정)를 기반으로 프롬프트를 체계적으로 작성하는 습관 들이기
 - 예: "중심 주제" + "맥락(시간, 장소, 감정)" + "색상 스타일" + "텍스처/배경 디테일"

<기본 템플릿 만들어서 활용하기>
 - 자주 사용하는 주제나 스타일에 맞춘 기본 프롬프트 템플릿을 만들어서 활용
 - 필요할 때 템플릿에 세부 내용만 추가해 빠르게 이미지 생성 가능
 - 예: "[맑고 활기찬] 분위기의 배경에서, [아이템]이 중심에 배치된 모습"

<반복 작업은 자동화, 창의적 작업에 집중>
- 반복적인 시도와 수정 작업은 AI에게 맡기고, 핵심 메시지와 창의적 아이디어에 집중하기
- 예: AI가 5~10개의 다양한 색상, 구성, 스타일을 자동 생성 → 결과물 비교 후 선택
- 예: 브랜드 컬러 테스트, 여러 색상 팔레트를 시도해 가장 적합한 색상 선택
